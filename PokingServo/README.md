# How to use

This project is meant to be built in [the Arduino IDE](https://docs.arduino.cc/software/) and is using the EEPROM.h library from Arduino and the forked VarSpeedServo.h library from [https://github.com/solo-fsw/VarSpeedServo](https://github.com/solo-fsw/VarSpeedServo). It is tested on an Arduino Leonardo.
