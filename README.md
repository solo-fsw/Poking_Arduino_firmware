This Arduino firmware was part of a closed source research project that was eventually conducted in a different way. To give others the chance to use this we made this public. Please note the hardware was never fully built and thus, never fully tested!

The firmware was intended to be used on an Arduino Leonarde in a device to 'poke' a participant with a servo, when the Arduino was told so via the serial port over USB.
